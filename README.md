## Part 2: Coding Technique Exploration
### p5.js – Vector Movement + Mouse Interaction
**Technique:** Using p5.js `p5.Vector` for smooth motion, `mouseX`/`mouseY` for cursor tracking, and frame-by-frame animation to create interactive particle trailing effects.

**Description (≤100 words):**
This technique uses vector math to calculate natural movement, making particles smoothly follow the mouse. It supports real-time interaction, dynamic animation, and visual trailing effects. It directly realizes the interactive particle style from Part 1, with high technicality and strong visual performance, perfect for creative coding projects.
