# Quiz 8 – Design Research

## Part 1: Imaging Technique Inspiration

### 🌊 *The Great Wave off Kanagawa* – Katsushika Hokusai

**Inspiration Source:** *The Great Wave off Kanagawa* (1831) by Katsushika Hokusai.

**Description (≤100 words):**

I am inspired by the dramatic composition and stylised wave forms in Hokusai’s *The Great Wave off Kanagawa*. The exaggerated curves and claw-like foam create a strong sense of motion and tension. I want to translate this visual language into an interactive system where waves are represented through dynamic particle motion. This technique is beneficial because it combines strong visual storytelling with abstraction, allowing me to create an expressive and interactive scene that aligns with the assignment’s focus on creative computational imagery.

## Part 2: Coding Technique Exploration

### ⚙️ Particle System (Interactive Simulation)

**Technique:** Particle system with mouse interaction controlling direction and emission.

**Description (≤100 words):**

A particle system can simulate the movement of water by representing wave foam as many small particles. Each particle has properties such as position, velocity, and lifespan, allowing complex motion to emerge from simple rules. By linking mouse movement to particle direction and using clicks to trigger bursts, the system can mimic the crashing and splashing effect of waves. This technique is effective because it enables dynamic, responsive visuals that capture the energy and flow seen in Hokusai’s composition.
