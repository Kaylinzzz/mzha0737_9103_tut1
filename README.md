# Quiz 8 – Design Research

## Part 1: Imaging Technique Inspiration

### 🧬 Reaction–Diffusion Pattern (Alan Turing Model)

**Inspiration Source:** Reaction–Diffusion systems (Turing Patterns), widely used in generative art and computational design.

**Description (≤100 words):**

I am inspired by the organic, self-evolving patterns generated by reaction–diffusion systems, where simple chemical rules produce complex, lifelike textures. These visuals resemble natural phenomena such as animal skin, coral growth, or cellular structures. I want to incorporate this technique into my project by creating an evolving, interactive surface that responds to user input. This approach is beneficial because it introduces emergent behaviour and complexity, aligning with the assignment’s goal of producing dynamic and visually engaging computational imagery beyond simple animation.

## Part 2: Coding Technique Exploration

### ⚙️ Gray–Scott Reaction Diffusion Algorithm

**Technique:** A numerical simulation of chemical interaction using grid-based computation (Gray–Scott model), often implemented with shaders or pixel buffers.

**Description (≤100 words):**

The Gray–Scott algorithm simulates two interacting chemicals diffusing and reacting over time, producing complex patterns from simple equations. In p5.js or WebGL, this can be implemented using pixel arrays or fragment shaders for real-time performance. This technique allows the creation of continuously evolving textures that feel organic and unpredictable. By linking parameters to user input (e.g., mouse interaction), the system becomes interactive. This directly supports my goal of generating dynamic, emergent visuals that go beyond predefined shapes and create a living visual system.
